# Tree of the plugin section
`tree ~/.vim/bundle -d -L 1`
```
├── nerdtree
├── pathogen
├── surround
├── tagbar
├── vim-airline
├── vim-airline-themes
├── vim-fugitive
├── vim-fullscreen
```
# Tips & tricks
1. To enable vim-fullscreen hit CTRL + ENTER in normal mode. Same to disable.
   Or hit `:FullScreenStart/Stop`
